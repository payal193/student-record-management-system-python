# Student Record Management System

A simple Python-based Student Record Management System that allows users to manage student records efficiently.

## Features

- Add Student Record
- View All Student Records
- Search Student by Roll Number
- Update Student Details
- Delete Student Record
- File-based Data Storage

## Technologies Used

- Python
- File Handling
- Functions
- Loops
- Conditional Statements
- Exception Handling

## Project Structure

```text
student-record-management-system-python/
│
├── main.py
├── students.txt
└── README.md
```

## How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/student-record-management-system-python.git
```

2. Navigate to the project folder

```bash
cd student-record-management-system-python
```

3. Run the program

```bash
python main.py
```
